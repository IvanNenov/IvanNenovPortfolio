C# Databases

MS SQL
The course introduces us to one of the most widely used database management systems (DBMS) used in the development of modern information systems - Microsoft SQL Server. The course focuses on relational modeling, data modeling with ER diagrams (tables and relational links) and SQL language (data retrieval, selection, projection, joining, aggregation, grouping, modification, deletion and insertion).
Entity Framework Core
The course tracks in detail the ORM technology: Entity Framework Core (EF core), which is an ORM standard in C # and .NET Core applications. EF Core allows mapping between a relational base and an object-oriented model using both the "database first" and "code first" approaches and provides a powerful object-oriented API for querying the database and performing CRUD operations.
